FAQ

What is NativeScript?

NativeScript is an open source framework for building native cross-platform mobile apps with JavaScript/TypeScript. One code base, multiple platforms, using the web skills you know and love. Learn more about the NativeScript framework.

How do I use CSS with a native mobile app?

NativeScript uses a subset of CSS for styling apps. The same CSS you use for web development; including id, class, and type selectors. Learn more about styling your NativeScript app.

The CSS I generate here isn't working with my app!

Make sure you are following the instructions provided when you download your theme (specifically, make sure you are using one of the core CSS themes along with the one you download). If you're pretty darn sure it's not user error, feel free to submit an issue on Github.


Download CSS Theme

1) Make sure your app is using the core themes introduced with NativeScript 2.4. Read more about the core themes and follow the instructions to verify you are using them properly.

2) You will be downloading a CSS file. This should be applied to your app after the core theme (since the CSS we generate supplements the core themes). More info here.

For reference, the core theme you are supplementing is the light theme.
